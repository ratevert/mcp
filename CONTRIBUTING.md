# Contributing

Thanks for helping improve Ratevert MCP.

1. Use Node.js 20 or newer and run `npm ci`.
2. Create a focused change with tests in `server.test.mjs` when behavior changes.
3. Run `npm test`, `npm run verify:release`, and `npm pack --dry-run`.
4. Do not add credentials, private Ratevert application code, deployment files,
   generated build output, or production data.
5. Keep the server anonymous, read-only, and safe to run behind a trusted proxy.

By contributing, you agree that your contribution may be licensed under the
repository's MIT License.
